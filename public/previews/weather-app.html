Weather App

The obligitory weather app!

Built using React with Webpack, the app sends API requests to the RapidAPI US weather service. Necessary login credentials are secured in the back-end using env variables and the project is currently hosted on Heroku.

Project to be converted to a React Native app.

PLEASE NOTE: this project is hosted on a free Heroku dyno. It will take approximately 30 seconds to load the live site.
