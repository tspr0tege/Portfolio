Quest Log

This project has seen several permutations already, and will see many more. Originally started as a two-day sprint MVP project, I decided to develop it out a little more. However, after returning to it some months later, I realized I could vastly improve it's entire structure and so I've set out to completely rewrite it from the ground up. What you see here, is the current progress as of 12/9/2021

Both the original and the rewrite are hosted on Heroku. The old one has much more put into design (even if it is still fairly amateurish), but the code in the new version is much more robust, well-written, and scalable. Also, for the rewrite I have decided to focus on one core aspect, instead of 3 at once.

Ultimately, this project will be deployed to AWS, and utilize a Mongo database. Currently, it's in a 'demo' production phase and can be used by anyone, with localStorage used to save input.
